Для создания голосовых файлов вы можете запустить сценарий сборки в mov/working/<br/>
Дополнительную информацию см. в [Readme](mov/working/README.md).

# Упаковка
Для упаковки мода можно использовать сценарии сборки в BUILD/<br/>
Просто запустите файл build.bat, который добавит файлы в соответствующие cat-файлы и создаст файлы SPK и ZIP.
- Либо воспользоваться альтернативной утилитой сборки озвучки, скачав ее по адресу https://apocalypse.moy.su/forum/362-2101-1
